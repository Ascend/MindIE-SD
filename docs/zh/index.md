# MindIE SD

MindIE SD 是面向昇腾的 Stable Diffusion 及相关多模态生成工作负载推理加速工具包，提供面向昇腾的融合算子、编译路径、缓存策略、量化特性和多卡执行能力，适用于生产级推理工作流。
